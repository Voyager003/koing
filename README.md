# Koing (코잉)

macOS 한영 자동변환 유틸리티. 영문 모드에서 한글을 잘못 입력하면 자동으로 변환합니다.

`rkskek` → `가나다` | `dkssud` → `안녕` | `gksrmf` → `한글`

## 설치

```bash
brew tap voyager003/koing
brew install --cask koing
```

또는 [Releases](https://github.com/Voyager003/koing/releases)에서 zip을 다운로드하세요.

## 권한 설정

시스템 설정 → 개인 정보 보호 및 보안 → 손쉬운 사용 → Koing 허용

## 사용법

| 단축키 | 기능 |
|--------|------|
| `⌥ Space` | 수동 변환 |
| `⌥ Z` | 되돌리기 |

타이핑 후 300ms 대기 시 자동 변환. 흔한 영어 단어는 변환하지 않습니다.
